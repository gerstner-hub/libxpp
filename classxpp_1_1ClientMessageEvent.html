Detailed Description

Wrapper around the XClientMessageEvent type.

A client message is not interpreted by the X server itself, but can contain e.g. window manager related information. The type field contains an atom that defines further content of the messsage.

Examples for this are the XEMBED extension and the WM_PROTOCOLS extension.
