Detailed Description

template<typename EVENT>
class xpp::InputEvent< EVENT >

Wrapper around input event types that mostly share the same fields..

Input handling events like XMotionEvent and XButtonEvent share most of the event fields. This templated base class allows to share code for these events while still providing type safety.

Specializations of this call need to be derived to make it useful
