Detailed Description

template<typename PROPTYPE>
class xpp::Property< PROPTYPE >

X11 property representation.

Based on the PropertyTraits definitions this class allows to have C++ Property objects that can get and set data transparently from/to the X server and transform the data from the native C++ world into the X world and vice versa with some type safety.

This type should only be used for temporarily wrapping actual program data / X11 data. The data should be processed right away or should be copied for long time storage. Especially string pointers are dangerous to be passed outside of a local function context.
