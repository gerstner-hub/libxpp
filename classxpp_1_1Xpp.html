void init ( std::optional< cosmos::ILogger * >  )
friend

Initializes the X++ library before first use.

The initialization of the library is required before any other functionality of libX++ is accessed. This initialization should occur after the main() function has been entered and not from within static initializers to avoid issues with static initialization order in executables / libraries.

Multiple initializations can be performed but finishLibXpp() needs to be called the same number of times for cleanup to occur.

Parameters
[in,out]loggerIf set then this Cosmos logger instance will be used for runtime error or debugging messages presenting internal library state.
